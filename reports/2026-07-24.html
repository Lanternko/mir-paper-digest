Pushing the Frontier of Full-Song Generation: Hierarchical Autoregressive Planning Meets Flow-Matching Rendering

提出歌詞／文字／屬性條件的全曲生成框架：hybird-LM 階層式自迴歸規劃 8-codebook RVQ token，FullDiT 在 VAE 連續潛空間做全曲 flow matching 渲染；支援純伴奏與保留旋律的翻唱，並以 DPO／GRPO／OPD 與 flow-based GRPO 做後訓練。

重要度

Problem

全長歌曲生成需同時處理歌詞條件、長程結構、人聲與伴奏音質，以及風格翻唱時的旋律保持；單一模型難以兼顧離散規劃效率與連續波形保真，後訓練訊號也常只打在其中一段。

Method

四元件架構：semantic-aware tokenizer 將音訊編成 8-codebook RVQ；hybird-LM 做階層式自迴歸全曲 token 規劃；FullDiT 在連續 VAE 潛空間以 codec token、歌詞與文字 caption 為條件做全曲 flow matching 渲染；兩層 melody module 從參考音抽離散旋律 cue 支援 cover。後訓練上對 hybird-LM 試 DPO／GRPO／OPD，對 FullDiT 用 flow-based GRPO。

Data

多語自動基準與 Artificial Analysis Music with Vocals leaderboard 評估；任務含 Lyrics-to-Song、Instrumental、Cover Song。細節以 tech report 為主。

Findings

在報告的多語自動評測與 leaderboard 設定下達 competitive 表現；顯示「AR 規劃 + flow 渲染 + reward 後訓練」可作為全曲生成的可擴充配方，並同時覆蓋歌詞條件、純伴奏與保旋律翻唱。

Limitations

tech report 型表述，資料規模、開源權重與日文／JPOP 歌詞細粒度對齊品質未充分展開；Automatic Analysis leaderboard 與 lab 內自動指標對唱詞可懂度／結構合法性的覆蓋仍需人工抽聽。封面旋律保持與 mean_audio 生成音的客觀對照亦未完整公開。

Is One Score Enough? Assessing Singing Quality of Songs with Temporal Score Curves

提出 SongSQA：全曲唱聲品質評估不只給一個分數，而產出時序片段品質曲線。先以教師模型偽標訓片段評分器，再用可學習 song embedding 與 self-attention 聚合，在多資料集上相對最強基線 KTAU 最高提升約 13.95%。

可轉譯 singing quality temporal score full-song SQA
Insight

洞見:karaoke-jp 與唱聲管線缺的常不是「再一個全域 MOS」，而是「哪一段唱砸了、哪一段可交付」。SongSQA 把 full-song SQA 做成 temporal score curve，對接 7/21 DTW 可靠度那條「片段級可信任」敘事：對齊有 confidence，唱聲品質也該有 per-segment 曲線。生成側可反過來當 MeanAudio／lyrics-to-song 的 reward 或 fail-loud 閘門。

Try next

可做:把 SongSQA 曲線掛在 karaoke 人聲 stem 上，低分段落對齊 LRC 行級時間戳做「需重錄／需修 pitch」標記；對全曲生成結果用曲線當 preference pair 的維度之一，對照單一 CLAP／FAD。

重要度

Problem

既有 Singing Quality Assessment 多針對短片段，不足以評估全長歌曲；全曲 SQA 需建模局部品質變化如何影響整體判斷，且片段級標註稀缺，若把整曲分數直接複製到每一段會把不同品質片段誤當等價。

Method

兩階段 SongSQA：Stage1 以預訓練教師產生偽標，訓練 Segment Score Predictor 做無人工片段標註的局部評分；Stage2 的 Song Quality Aggregator 融合片段特徵與預測分數，以可學習 song embedding 加 self-attention 捕捉片段表現與整曲品質的連結，同時輸出整體分與時序品質曲線。

Data

多個 singing quality 資料集；報告相對最強基線 KTAU 最高相對提升 13.95%，並在其他指標上一致提升。

Findings

片段偽標 + 注意力聚合能有效做全曲 SQA，並提供可解釋的 temporal score curve；顯示「單一總分」不足以描述唱聲表現的時間結構。

Limitations

片段監督仍依賴教師偽標，教師偏差會傳到曲線；摘要未詳述日文／JPOP 語料覆蓋、伴奏殘留與多語言歌詞條件。與 pitch／節奏錯誤類型的對齊解釋力需另驗。

Editor Note

karaoke-jp 今日首選：把唱聲評估從「一個分」升級成「可定位的時序曲線」。可與 DTW 可靠度、LRC 修齊工具串成「分離 → 對齊 → 品質曲線 → 人工／重生成」。MeanAudio 側可當生成唱聲的片段 reward。
